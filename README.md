# ALCHEMIA

A browser-based alchemy game. Combine all 118 periodic table elements to discover hundreds of new substances.

## How to play

Open `Alchemia.html` in a browser — no build step, no server required.

Drag elements from the library onto the canvas and drop them on top of each other to combine. New discoveries unlock more recipes. Your canvas and progress are saved automatically between sessions.

## Features

- 118 real periodic table elements with atomic data
- ~1 500 procedural recipes across multiple tiers
- Pixel-art icons for every element (14×14, multiframe)
- Ambient soundscape that evolves as you discover more
- Library with search, category filters (ALL / METALS / GASES / ORGANICS / RECENT), and a COMBOS tab sorted by input availability
- Hint ticker with clickable icons that spawn elements directly onto the canvas
- Canvas state persisted to `localStorage` — restored on next visit, Water + Fire pre-spawned on first visit
- Drag-to-position from library; golden-angle spiral spawn for tap/click
- First-discovery glow ring, tier-unlock toasts, progress bar milestones
- Inline reset confirmation with auto-cancel
- Game completion screen at 100% discovery

## Project structure

```
Alchemia.html          entry point — loads scripts in order
styles.css             all CSS (reset, layout, animations, responsive)

app.jsx                root App component, game state, audio engine, useAmbient hook
components/
  pixel-icons.jsx      PixelIcon, PixelArrowUp, PixelArrowDown
  top-bar.jsx          TopBar — discovery counter, HELP, CLEAR with divider
  play-area.jsx        PlayArea, Tile — canvas, drag/drop, empty-state animation
  library.jsx          Library, LibCard — elements + combos tabs, category filter strip
  hint-ticker.jsx      HintTicker — hint strip with refresh button and spawn-on-tap
  toasts.jsx           Toasts — dismissible, queued, pixel SVG spark
  help.jsx             Help — rules panel

icons.js               procedural pixel-art icon renderer (14×14 grid, multiframe)
recipes.js             element metadata + recipe database (~1 500 recipes)
engine.js              canvas FX — ambient glow + particle bursts
periodic-elements.js   all 118 periodic table elements with atomic data
tweaks-panel.jsx       dev-only tweaks panel (sound, particles, FPS)

dist/                  minified build output (generated by npm run build)
scripts/
  inject-version.js    stamps a cache-busting hash into dist/ asset references
```

## Tech

No bundler at runtime. React 18 + Babel loaded from CDN, JSX compiled in-browser. Plain JS for game logic. `npm run build` produces a minified `dist/` via Babel + Terser + clean-css for deployment.

## Build

```
npm install
npm run build
```

Output lands in `dist/`. `Alchemia.html` points to `dist/` assets when served; the raw source files are used for local development.
